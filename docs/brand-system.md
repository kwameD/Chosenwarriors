# Chosen Warriors Brand System

## Colors

- Primary black: `#000000`
- Secondary black: `#1A1A1A`
- White: `#FFFFFF`

The website now uses a strict black-and-white visual system. Black carries the primary brand weight for navigation, buttons, section headers, icons, and dark bands. White is used for page backgrounds, cards, buttons on dark sections, and readable contrast.

## Typography

- Primary typeface: Poppins
- Fallback stack: Inter, system UI, sans-serif
- Hero titles use bold, high-contrast display sizing.
- Section headings use consistent `section-title` sizing.
- Labels use uppercase treatment for scannable ministry categories.

## Usage

- Use black for primary actions, active navigation, section identifiers, icons, and dark content bands.
- Use white for backgrounds, reversed text, and action buttons placed on black sections.
- Use borders, opacity, shadow, and spacing instead of additional accent colors.
- Keep cards at an 8px radius and reserve them for grouped content, forms, previews, and repeated items.
